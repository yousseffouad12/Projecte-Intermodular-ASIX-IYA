# 🚀 Projecte Intermodular -- ASIX

## 👨‍💻 Qui som?

Som estudiants de **2n curs del Cicle Formatiu de Grau Superior
d'Administració de Sistemes Informàtics en Xarxa (ASIX)** i estem
desenvolupant el nostre **Projecte Intermodular Final**.

Aquest projecte representa l'aplicació pràctica de tots els coneixements
adquirits al llarg del cicle formatiu, integrant diferents mòduls i
tecnologies en una solució real.
